## Getting Started

### Logging In

1. Navigate to the Afrithrive login page.
2. Enter your username and password.
3. Click "Login".
4. If it's your first login, you may be prompted to change your password.

### Navigation

- Use the sidebar menu to navigate between different modules.
- The menu items you see depend on your assigned permissions.
- Click on any menu item to access that module.

### User Profile

- Click on your profile icon to access personal settings.
- Update your information, change password, or manage preferences.

## Dashboard

The Dashboard provides an overview of key metrics and activities.

### Features

- **Statistics Cards**: View counts of active jobs, scholarships, users, etc.
- **Recent Activities**: See latest job applications, scholarship submissions, etc.
- **Charts and Graphs**: Visual representations of data trends.
- **Quick Actions**: Direct links to create new items or view recent entries.

### How to Use

1. After logging in, you'll land on the Dashboard.
2. Review the statistics to understand system activity.
3. Use the quick action buttons for common tasks.
4. Click on chart elements for detailed views.

## Jobs Management

Manage job postings and opportunities.

### Creating a Job

1. Navigate to "Jobs" in the sidebar.
2. Click "Add New Job" or similar button.
3. Fill in job details:
   - Title
   - Description
   - Requirements
   - Salary range
   - Location
   - Application deadline
4. Set status (Draft, Published, Closed).
5. Save the job.

### Editing Jobs

1. Go to the Jobs list.
2. Click on a job to view details.
3. Click "Edit" to modify information.
4. Save changes.

### Managing Job Status

- **Draft**: Not visible to applicants.
- **Published**: Active and accepting applications.
- **Closed**: No longer accepting applications.

## Job Applications

Review and manage job applications.

### Viewing Applications

1. Go to "Job Applications".
2. Filter by job, status, or date.
3. Click on an application to view details.

### Processing Applications

1. Open an application.
2. Review applicant's information and attachments.
3. Update status:
   - Pending Review
   - Shortlisted
   - Interview Scheduled
   - Rejected
   - Hired
4. Add notes or feedback.
5. Communicate with applicants if needed.

### Bulk Actions

- Select multiple applications for bulk status updates.
- Export application data for reporting.

## Scholarships

Manage scholarship programs.

### Creating Scholarships

1. Navigate to "Scholarships".
2. Click "Add New Scholarship".
3. Enter details:
   - Title
   - Description
   - Eligibility criteria
   - Award amount
   - Application deadline
   - Required documents
4. Set status and save.

### Managing Scholarships

- Edit scholarship details.
- View application counts.
- Close scholarships when deadlines pass.

## Scholarship Applications

Process scholarship applications.

### Reviewing Applications

1. Go to "Scholarship Applications".
2. Filter applications by scholarship or status.
3. Open applications to review documents and information.

### Decision Making

1. Evaluate based on criteria.
2. Update status:
   - Under Review
   - Approved
   - Rejected
   - Awarded
3. Provide feedback to applicants.

## Payments

Handle financial transactions.

### Viewing Transactions

1. Navigate to "Payments".
2. View transaction history.
3. Filter by date, type, or status.

### Processing Payments

1. For pending payments, review details.

### Reporting

## Opportunity Requests

Handle user requests for opportunities.

### As a User

1. Submit a request for job/scholarship information.
2. Provide details and attach documents.
3. Track request status.

### As an Administrator

1. Review incoming requests.
2. Mark as "Received" to acknowledge.
3. Research and provide opportunity details.
4. Mark as "Completed" with full information.

### Workflow

- **Pending**: New request.
- **Received**: Acknowledged by admin.
- **Completed**: Response provided with opportunity details.

3. Manage email templates.

## Troubleshooting

### Common Issues

**Can't access a module**: Check with your administrator about your permissions.

**Login problems**: Reset password or contact support.

**Data not saving**: Ensure all required fields are filled.

**Slow performance**: Clear browser cache or check internet connection.

### Getting Help

- Use the support link in the sidebar footer.
- Contact your system administrator.
- Check in-app notifications for updates.

- **Support Hours**: 24/7
- **Contact**: Visit Support Center (link in sidebar)
- **Email**: support@afrithrive.com
